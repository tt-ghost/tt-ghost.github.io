常见问题

你可能想了解的

关于文流的使用、功能与付费的常见疑问。

文流是否需要注册账号？

不需要。文流是一款纯本地应用，无需注册、无需登录，下载即可使用。你的所有数据仅保存在你的设备上。

AI 功能如何使用？需要付费吗？

你需要在应用设置中添加 AI 服务商的 API Key（如 DeepSeek、OpenAI 等）。免费版每月有基础 AI 调用额度，超出后可升级 Pro 或购买加油包。API 调用费用由服务商直接收取，文流不收取额外费用。

我的文章数据安全吗？

非常安全。所有文章和灵感碎片都存储在你的设备本地 SQLite 数据库中，不会上传到任何服务器。API Key 使用 iOS Keychain 加密存储。AI 调用时的文本直接从你的设备通过 HTTPS 发送至第三方 API，文流无法访问。你可以随时导出或删除你的数据。

支持的 AI 模型有哪些？

文流支持所有兼容 OpenAI Chat Completions API 的服务商，内置了 DeepSeek、OpenAI、Google Gemini、智谱 AI 等常用服务商的配置模板，你也可以手动添加其他服务商或自部署模型。

写好的文章如何发布到公众号？

在编辑器中点击右上角菜单，选择"复制为公众号格式"，系统会将文章内容（包括排版和图片）以公众号兼容的 HTML 格式复制到剪贴板。打开微信公众平台编辑器粘贴即可，图片会自动转为 base64 内嵌格式。

Pro 订阅如何取消？

Pro 订阅通过 Apple ID 管理。在 iOS 设备上进入"设置 > Apple ID > 订阅"，找到文流即可取消自动续费。当期结束前取消，当期内仍可享受 Pro 权益。删除应用不会自动取消订阅，请通过上述路径手动操作。

文流是否有 Android 版本？

目前文流仅支持 iOS 17 及以上版本。我们专注于在 Apple 平台上提供最佳的写作体验。未来是否会推出 Android 版本取决于用户需求和开发资源。

如何备份我的文章？

建议通过以下几种方式备份：1) 使用 iCloud 备份，文章数据会随设备备份一同保存；2) 通过 iTunes / Finder 将设备数据备份到电脑；3) 定期使用"复制为公众号格式"将重要文章导出到其他平台或本地文件。
